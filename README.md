## Installation

You can install the package via composer:

```bash
composer require mohamedfathim/unit-conversions
```

## Usage

```php
$skeleton = new WeightConversion\UnitConversions();
echo $skeleton->echoPhrase('Hello, WeightConversion!');
```

## Testing

```bash
composer test
```
